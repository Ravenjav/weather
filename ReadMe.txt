Взамодействие:

1) Запрос /takeWeather - получаете последнюю в бд информацию о погоде (Локация: Минск, Время обновления 1 час).
Время обновления указано в статической перменной Time(интервал в минутах) в классе AsinAppService.
2) Запрос /takeAvgTemperature - для самого запроса нужно, также, в Body передать две переменных (from и to).
 ! Важно, для удобства, формат, вводимой даты, был изменен на YYYY-MM-DD. При вводе не валидных данных придет
  massage с указанием примерной области ошибки. Если на полученный временной промежутак нет полной информации,
  то будут выведены только те даты, которые есть в БД.
